# BarberBot 💈

**BarberBot** — это Telegram-бот для онлайн-записи и регистрации клиентов в барбершоп. Бот ведет интерактивный диалог с пользователем, собирает необходимые данные (имя, телефон, желаемая дата и время) и сохраняет их в базе данных.

## 🚀 Возможности (Features)
- 📝 **Регистрация клиентов:** Пошаговый сбор данных для записи на стрижку (имя, номер телефона, дата и время).
- 🗂 **Просмотр своих данных:** Пользователь может просмотреть информацию о своей текущей записи по команде `/mydata`.
- ❌ **Отмена записи:** Возможность отменить запись и удалить свои данные с помощью команды `/cancel`.
- 💾 **Длительное хранение:** Временные состояния хранятся в JSON файле (`user.json`), а подтвержденные записи — в надежной реляционной базе данных **SQLite**.

## 🛠 Стек технологий
- **Язык:** [Python 3](https://www.python.org/)
- **Фреймворк:** [Aiogram 3.x](https://docs.aiogram.dev/) (Асинхронная работа с Telegram Bot API)
- **База данных:** [SQLite](https://www.sqlite.org/) + [SQLAlchemy 2.0](https://www.sqlalchemy.org/) (ORM)
- **Переменные окружения:** python-dotenv

## 📂 Структура проекта
```text
BarberBot/
├── zapisbot.py     # Главный файл для запуска Telegram-бота
├── models.py       # SQLAlchemy модели (таблица users)
├── database.py     # Настройка подключения к БД и сессий SQLAlchemy
├── requests.py     # CRUD операции для взаимодействия с базой данных
├── user.json       # JSON-файл для отслеживания текущих шагов (state) пользователей
├── .env            # Файл с секретными переменными окружения (не добавляется в git)
└── users.db        # Файл базы данных SQLite (создается автоматически)
```

## ⚙️ Установка и запуск

### 1. Клонирование репозитория
Откройте терминал и выполните команду:
```bash
git clone https://github.com/ваш-логин/SignUpBot.git
cd SignUpBot
```

### 2. Создание виртуального окружения (Рекомендуется)
```bash
python -m venv .venv
source .venv/bin/activate  # Для Linux/macOS
# или
.venv\Scripts\activate     # Для Windows
```

### 3. Установка зависимостей
Убедитесь, что у вас установлены нужные библиотеки:
```bash
pip install aiogram sqlalchemy python-dotenv groq
```
*(Если в проекте появится файл `requirements.txt`, используйте: `pip install -r requirements.txt`)*

### 4. Настройка переменных окружения
Создайте в корне проекта файл `.env` и добавьте в него ваш токен Telegram бота, а также строку подключения к базе данных.
```env
bot_token=ВАШ_TELEGRAM_BOT_TOKEN
DATABASE_URL=sqlite:///users.db
# ai_key=ВАШ_КЛЮЧ_GROQ_ЕСЛИ_ИСПОЛЬЗУЕТСЯ
```
*Токен бота можно получить в Telegram у [@BotFather](https://t.me/BotFather).*

### 5. Запуск бота
Выполните главную точку входа:
```bash
python zapisbot.py
```
Если всё сделано верно, в консоли появится сообщение: `Бот запущен...`

## 💬 Доступные команды бота
- `/start` — Приветственное сообщение и начало работы.
- `/register` — Начать процесс записи: бот поочередно запросит имя, телефон и время.
- `/mydata` — Показать оформленную запись.
- `/cancel` — Отменить оформленную запись и удалить данные.
